What is IONIC?

Ionic is a free, open source framework for building mobile apps. Developers use web technologies like HTML, CSS, and JS to build mobile apps in the Ionic framework.

Visit ionic's website or peep their docs when you get the chance. There're some cool things there.


Advantages of Using Ionic

  • Efficient for prototyping
  • Developers who do not know Swift/Java/Objective C can still build mobile apps as long as they know how to build web apps
  • Can emulate an iOS/Android environment for testing
    • Can also deploy code (written once) to different platforms
  • Ionic syntax is similar to HTML/JS
  • Live reload

Disadvantages of Using Ionic

  • Additional features cost money
  • The tutorial parts of their documentation are not beginner friendly
  • Requires many node packages to be installed in order to use

Ye Old Native App v. Web App Battle

Native Apps Web Apps
  • Built on programming languages like Objective-C and Java
  • Installed via app stores
  • Good at accessing phone data
  • Push notifications access
  • Work offline and in the background
  • Faster than web apps
  • Built on languages like Javascript, CSS, and HTML
  • Live on browsers
  • Cheaper to develop
  • More access to resources like APIs
  • Scalable across many platforms

Introducing...the HYBRID APP

Hybrid apps combine the power of native and web apps to reap the benefits of both. They use a NATIVE infrastructure but contain WEB-BASED content. Think of them as apps with a browser embedded inside. There are, however, some cons of hybrid apps (nothing is perfect):

  • Games run better on native apps
  • Slower performance unless optimized
  • May experience web-based glitches

Two Best Buds, Ionic & Bootstrap

Ionic Online Creator

Don't know how to code? You can still use Ionic using their online drag and drop creator! It is customizable and super easy to use. It's really useful for prototyping and visualizing what your app will look like on an iOS or Android device.

Ionic Command Line Interface (CLI)

If you're a more experienced programmer, you may be interested in using Ionic's command land. It gives you more flexibility for design than the creator. You can also link backend scripts to your code and customize the Javascript your app currently uses. Some crucial Ionic commands to know:

  • ionic start
  • ionic serve
  • ionic build
